"@angular/cli": "~1.7.4",

## Deployed website

[https://sociallappp.herokuapp.com/](https://sociallappp.herokuapp.com/)

## Features

* SignUp / SignIn.
* Forgot / Reset password.
* create / edit / delete Clients.
* create / edit / delete Invoices.
* View and Download invoice in pdf format.
